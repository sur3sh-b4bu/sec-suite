🔴 Template Math Discovery

🔴 Jinja2 Class Escalation

                    
Using Python's MRO (Method Resolution Order) to walk from a string class up to the
                        object class, then down to dangerous subclasses like subprocess.Popen.
                    

                

                

                    
🟡 Twig Filter Execution

                        exec or system) as filters and then calling them.

                        malicious files directly to the web root.
