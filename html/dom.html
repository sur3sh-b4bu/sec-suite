DOM Vulnerability Index

🔴 DOM XSS

Occurs when an application contains client-side JavaScript that processes data from an untrusted source in an unsafe way, usually by writing the data back to the DOM.

🔴 Open Redirect (DOM)

Triggered when data from a source is used to modify the location property of the current window, allowing attackers to redirect users to malicious domains.

🟡 Web Storage Poisoning

Injecting malicious data into localStorage or sessionStorage that is later read and processed by a dangerous sink.

🟡 client-side SQLi

Occurs when an application uses a client-side database (like Web SQL) and incorporates untrusted data into a query in an unsafe way.
