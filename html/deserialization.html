Deserialization Research Index

            

                

                    
🔴 Magic Method Injection

                    
Triggering unintended logic execution in PHP __wakeup, __destruct, or
                        __toString during object recreation.

                

                

                    
🔴 Java Gadget Chains

                    
Leveraging existing classes in the classpath to form a chain of method calls that ultimately
                        leads to arbitrary command execution.

                

                

                    
🟡 Python Pickle Native

                    
The pickle module is fundamentally insecure. Deserializing untrusted data leads to
                        direct RCE via __reduce__.

                

                

                    
🟡 YAML Unsafe Load

                    
Exploiting default unsafe loaders in YAML parsers to instantiate arbitrary classes and trigger
                        code execution.
