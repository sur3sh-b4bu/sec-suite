GraphQL Research Index

            

                

                    
🔴 Introspection Discovery

                    
Leveraging internal __schema queries to map the entire API topology, including
                        hidden mutations and administrative objects.

                

                

                    
🔴 IDOR via Argument Fuzzing

                    
Manipulating identifier arguments (e.g., user(id: $id)) to access objects outside of
                        the current authenticated scope.

                

                

                    
🟡 Alias-based Batching

                    
Bypassing rate limits or MFA by combining hundreds of distinct queries into a single HTTP request
                        using aliased fields.

                

                

                    
🟡 Complexity Exhaustion

                    
Crafting deeply nested queries that force the server to execute an exponential number of database
                        lookups, leading to DoS.
