XSS Research Index

            

                

                    
🔴 Reflected Context

                    
Testing how input is mirrored in the HTML body. Payloads like
                        <script>alert(1)</script> exploit direct injection when no output
                        encoding is applied.
                    

                

                

                    
🔴 Attribute Breakout

                    
Injecting into HTML attributes (e.g., value="..."). Payloads like
                        " onmouseover="alert(1) breakout of the attribute and register an event handler.
                    

                

                

                    
🟡 DOM Sink Audit

                    
Exploiting cases where client-side JavaScript takes data from a source (like
                        location.search) and passes it to a dangerous sink (like
                        element.innerHTML).
                    

                

                

                    
🟡 SVG Image Injection

                    
Using SVG files to carry XSS payloads. Since SVG is XML-based, it can contain
                        <script> tags that execute when the image is rendered directly in the
                        browser.
