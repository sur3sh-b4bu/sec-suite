SQLi Research Index

            

                

                    
🔴 Error-Based Leakage

                    
Triggering specific database errors (e.g., via 1/0 or type conversion) that contain
                        data from the database within the error message itself.

                

                

                    
🔴 SQL UNION Selection

                    
Appending a UNION SELECT statement to the original query to retrieve data from other
                        tables in the database within the original output stream.

                

                

                    
🟡 Time-Blind Inference

                    
Submitting queries that use pg_sleep(5) or WAITFOR DELAY '0:0:5' to
                        infer information based on how long the server takes to respond.

                

                

                    
🟡 Out-of-Band Exfiltration

                    
Using database features to make network requests (e.g., DNS lookups) containing data as part of
                        the hostname, to bypass traditional response-based filtering.
