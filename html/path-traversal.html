Path Traversal Research Index

            

                

                    
🔴 dot-dot-slash Attack

                    
Using the ../ sequence to step up from the intended directory and access restricted
                        sensitive files on the parent filesystem.

                

                

                    
🔴 Filter-Recursive Bypass

                    
Using sequences like ....// to bypass simple non-recursive string replacement
                        filters that only strip a single ../ instance.

                

                

                    
🟡 Null Byte Truncation

                    
Appending %00 to the end of a path to terminate the string early, potentially
                        bypassing mandatory file extension checks (e.g., .jpg).

                

                

                    
🟡 Absolute Root Injection

                    
Providing the full absolute path (e.g., /etc/passwd) instead of a relative one,
                        exploiting cases where the application doesn't enforce a relative base.
