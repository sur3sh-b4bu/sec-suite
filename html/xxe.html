XXE Injection Research Index

            

                

                    
🔴 Classic Entity Injection

                    
Defining an external entity within the DOCTYPE that points to a local file (e.g.,
                        SYSTEM "file:///etc/passwd") and then referencing that entity within the XML body
                        to disclose content.

                

                

                    
🔴 Blind OOB Exfiltration

                    
Using an external DTD hosted on an attacker-controlled server to define a parameter entity that
                        extracts data and sends it back to the attacker via an HTTP request (URL-encoded).

                

                

                    
🟡 XInclude Exploitation

                    
Leveraging the xi:include element to include external files or resources when the
                        traditional DOCTYPE declaration is blocked or restricted by the XML parser's configuration.

                

                

                    
🟡 SVG-XXE Discovery

                    
Testing image upload functionality by submitting SVG (Scalable Vector Graphics) files containing
                        malicious XML entity definitions that the server-side image processor may evaluate.
