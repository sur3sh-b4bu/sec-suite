NoSQL Injection Research Index

            

                

                    
🔴 Operator Injection

                    
Submitting query operators ($ne, $gt, $in) as object
                        values to change the logic of the database filter.

                

                

                    
🔴 Login Logic Bypass

                    
Forcing an identity match by providing a {"$gt": ""} condition for the password
                        field, which is true for any non-empty string.

                

                

                    
🟡 Regex-based Extraction

                    
Using $regex to test potential password characters one-by-one, binary searching
                        through the authentication response code.

                

                

                    
🟡 Server-side JavaScript

                    
Exploiting the $where operator to execute arbitrary JavaScript on the MongoDB server
                        for data extraction or DoS.
