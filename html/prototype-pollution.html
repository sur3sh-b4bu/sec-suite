Prototype Pollution Research Index

            

                

                    
🔴 Client-side Pollution

                    
Manipulating URL parameters like __proto__[color]=red to inject properties into all
                        JavaScript objects, potentially leading to DOM-XSS.

                

                

                    
🔴 Server-side RCE

                    
Polluting the global Object.prototype on the backend to manipulate template engine
                        configuration (e.g., EJS) for arbitrary shell execution.

                

                

                    
🟡 Prototype Gadget

                    
Finding existing code ("gadgets") that use a property in an insecure way, which becomes
                        exploitable once that property is polluted globally.

                

                

                    
🟡 Constructor Evasion

                    
Bypassing __proto__ filters by using alternative paths like
                        constructor.prototype.targetProperty to achieve the same result.
