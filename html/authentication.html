Authentication Research Index

            

                

                    
🔴 Brute-Force Vulnerability

                    
Testing if an endpoint permits an unlimited number of login attempts, allowing for exhaustive
                        password guessing without account lockout or rate limiting.

                

                

                    
🔴 Broken MFA Logic

                    
Testing if the multi-factor authentication step can be bypassed by directly accessing the
                        application homepage or bypassing the MFA verification endpoint.

                

                

                    
🟡 Credential Reflection

                    
Testing if credentials or sensitive session tokens are reflected in the URL (GET parameters) or
                        stored insecurely in the browser's local storage.

                

                

                    
🟡 Account Enumeration

                    
Detecting whether the application response varies when a valid username is provided versus an
                        invalid one, facilitating targeted brute-force attacks.
