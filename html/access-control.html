Access Control Research Index

            

                

                    
🔴 IDOR (Direct Object Reference)

                    
Testing if changing a predictable ID in a URL or payload (e.g., user_id=1001 to
                        user_id=1002) allows unauthorized viewing or modification of another user's data.
                    

                

                

                    
🔴 Vertical Privilege Escalation

                    
Attempting to access administrative or restricted functions by manipulating identity parameters
                        (e.g., setting is_admin=true) or accessing hidden API nodes.

                

                

                    
🟡 Method Override Logic

                    
Testing if a restricted endpoint (e.g., DELETE) can be reached by using a permitted
                        method (e.g., POST) with headers like X-HTTP-Method-Override: DELETE.
                    

                

                

                    
🟡 Multi-Step State Bypass

                    
Testing if critical steps in a process (e.g., payment verification) can be skipped by directly
                        calling the subsequent step's API endpoint (e.g., order confirmation).
