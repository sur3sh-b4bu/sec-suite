OAuth Research Index

            

                

                    
🔴 Redirect URI Collision

                    
Manipulating the redirect_uri parameter to point to an attacker-controlled endpoint,
                        leaking the authorization code or access token.

                

                

                    
🔴 Missing State Validation

                    
Executing a CSRF attack by forcing a victim to link their account to an attacker's identity due
                        to non-existent or predictable state values.

                

                

                    
🟡 Fragment Leakage

                    
Exploiting the Implicit Grant type where tokens reside in the URL fragment (#),
                        making them accessible via malicious scripts or referrer headers.

                

                

                    
🟡 Dynamic Client Registration

                    
Registering an arbitrary malicious client on the OAuth server to gain access to system-level
                        scopes and user data.
