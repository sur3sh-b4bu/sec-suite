SSRF Research Index

            

                

                    
🔴 Localhost Bypass

                    
Accessing the loopback interface via obfuscated payloads like http://127.1 or
                        decimal values to reach local admin panels and internal services.

                

                

                    
🔴 Cloud Metadata Leak

                    
Querying the non-routable address 169.254.169.254 to extract high-value IAM
                        credentials and instance configuration from AWS, GCP, or Azure.

                

                

                    
🟡 Protocol Redirection

                    
Leveraging gopher:// or file:// to move beyond HTTP, potentially
                        interacting with redis, memcached, or reading local sensitive files.

                

                

                    
🟡 DNS Rebinding

                    
Controlling a DNS server that returns a public IP initially, then quickly switches to a private
                        local IP to bypass domain-based IP validation filters.
