Command Injection Research Index

            

                

                    
🔴 Separator Injection

                    
Using logical operators (;, &, |) to append malicious
                        commands to legitimate OS execution calls.

                

                

                    
🔴 Blind OOB Discovery

                    
Triggering external network requests (DNS/HTTP) from the victim server to confirm execution when
                        output is hidden.

                

                

                    
🟡 Time Correlation

                    
Using sleep or ping counts to infer execution success based on server
                        response latency.

                

                

                    
🟡 Filter Obfuscation

                    
Bypassing WAF/IDS limits using globbing, environment variables, or encoded character streams.
